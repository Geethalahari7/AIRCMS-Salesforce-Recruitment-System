# AIRCMS - AI Recruitment Management System

## Overview
AIRCMS is a Salesforce-based Recruitment Management System that automates the hiring process using custom objects, Flows, Reports, and Dashboards.

## Features
- Candidate Management
- Job Opening Management
- Application Tracking
- Interview Management
- Offer Management
- Automated Recruitment Flows
- Interactive Reports
- Recruitment Dashboard

## Technologies
- Salesforce Lightning
- Salesforce Flow Builder
- Reports & Dashboards
- Custom Objects
- Salesforce DX

## Custom Objects
- Candidate
- Application
- Interview
- Job Opening
- Offer

## Flows
- Application Auto Screening
- Interview to Offer
- Offer Approval Flow

## Reports
- New Candidates Report
- New Applications Report
- New Interviews Report
- New Job Openings Report
- New Offers Report

## Dashboard
AIRCMS Dashboard for Recruitment Analytics.
