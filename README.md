# AIRCMS - AI Recruitment & Candidate Management System

## 📌 Project Overview

AIRCMS is a Salesforce-based AI Recruitment & Candidate Management System that automates candidate management, job openings, applications, interviews, offers, reports, and dashboards.

---

## ✨ Features

- Candidate Management
- Job Opening Management
- Application Tracking
- Interview Management
- Offer Management
- Salesforce Flows Automation
- Reports & Dashboards
- AI Candidate Score
- Hiring Recommendation

---

## 🛠️ Technologies Used

- Salesforce Lightning
- Salesforce DX
- Flow Builder
- Reports & Dashboards
- Git
- GitHub

---

## 📂 Custom Objects

- Candidate
- Job Opening
- Application
- Interview
- Offer

---

## ⚡ Flows

- Application Auto Screening Flow
- Interview to Offer Flow
- Offer Flow

---

## 📊 Reports & Dashboard

- Candidate Report
- Application Report
- Interview Report
- Job Opening Report
- Offer Report
- AIRCMS Dashboard
